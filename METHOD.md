# Method

Geometry is treated as a candidate coordinate system.

The project asks whether geometric templates help organize findings in ways
that are more useful than ordinary outlines or flat lists.

## Template Test

For each candidate template, ask:

- Does it compress the findings?
- Does it predict missing relations?
- Does it survive new evidence?
- Does it distinguish unlike claims?
- Does it generate better next questions?
- Can it fail?

## Layer Separation

Public artifacts should keep four layers separate:

1. Source claim
2. Interpretive finding
3. Geometry mapping
4. Scientific status

If a mapped pattern cannot preserve those layers, the pattern fails.

## Initial Dynamic Model

The first candidate moving structure is a hybrid:

- radial distance: evidential distance from testable observation
- angular sector: domain of claim
- spiral overlay: developmental or temporal sequence
- edge tension: unsupported leap, contradiction, or uncertainty
- node weight: confidence
