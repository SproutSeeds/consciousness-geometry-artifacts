# Consciousness Geometry Artifacts Instructions

## Scope

This file governs the public artifact stream repo.

## Public Boundary

Everything here should be safe to publish.

Do not include private notes, raw full transcripts, credentials, local-only
scratch artifacts, or unpublished sensitive material from the private lab.

## Claim Discipline

Every artifact should distinguish:

- what a source says
- what we infer from it
- how we map it
- what evidence status it actually has

Use this sentence when needed:

> This artifact summarizes or maps source material. It does not validate the
> source claims.
